# Examples

This directory holds small examples for `swift-package-docs`.

Use examples to show:

- minimal target trees for Swift package repositories
- standard target trees for normalized repositories
- audit findings against the role model
- before and after normalization outcomes

Current examples:

- `standard-target-tree.md`: standard normalized end-state by role.
- `minimal-export-preview.md`: manual preview of the `minimal` profile output.
- `standard-export-preview.md`: manual preview of the `standard` profile
  output.
- `proposal-in-architecture.md`: proposal content misplaced in current-truth
  docs, then normalized into Proposal and Truth.
- `route-in-readme.md`: task-driven route language misplaced in a `README`,
  then normalized into `AGENTS.md` and index text.

Keep examples:

- small
- concrete
- tied to scaffold, audit, normalize, or export template

Do not turn examples into feature specs, release playbooks, or task systems.
