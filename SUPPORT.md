# Support

## Questions

For usage questions or general guidance, open a GitHub issue and describe the
repository area or skill involved.

## Bugs

Use the bug issue template for problems with:

- the `swift-skills` repository itself
- a specific skill
- repo-local docs, templates, profiles, or previews

## Feature Requests

Use the feature issue template for:

- new skills
- rule, template, or profile improvements
- repo-local governance or documentation improvements

## Security Issues

Do not report security-sensitive issues through normal bug reports. Follow
`SECURITY.md` instead.
