# Code of Conduct

## Our Pledge

We want `swift-skills` to be a respectful, constructive, and welcoming project
for everyone participating in the repository.

## Expected Behavior

- Be respectful in technical and non-technical discussions.
- Focus criticism on ideas, code, documentation, and behavior, not people.
- Assume good intent, ask clarifying questions, and keep feedback concrete.
- Respect differing viewpoints, experience levels, and communication styles.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, deliberate intimidation, or sustained disruption
- Publishing private information without permission
- Abusive, insulting, or demeaning language

## Scope

This code of conduct applies to repository discussions, issues, pull requests,
reviews, and other project spaces maintained for `swift-skills`.

## Reporting

If you experience or witness behavior that violates this code of conduct, open
a private or minimally scoped issue requesting maintainer follow-up. Do not
include sensitive personal details in a public report.

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull
requests, code, and other contributions that are not aligned with this code of
conduct.
